# Carding Genie - Web Application

Carding Genie is a software designed to make carding automated through Artificial Intelligence, created by caders for caders.

## Features

- User authentication (login/signup)
- Dashboard with various tools and functions
- Cryptocurrency funding options
- Multiple pricing plans
- Real-time balance tracking
- Responsive design for mobile and desktop

## Tech Stack

- Frontend: React (Vite)
- Styling: Tailwind CSS
- Backend: Supabase
- Authentication: Supabase Email/Password
- Database: Supabase PostgreSQL
- Icons: lucide-react
- Deployment: Vercel
